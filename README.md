# GBeauty Inventory Tracker

GBeauty Inventory Tracker is an inventory tracker for GBeauty beauty supply store.

# Sprint 1 Purpose

This is the client-side prototype. It uses HTML, CSS, and JavaScript to create an interactive browser-based prototype.

# Sprint 1 Features

- Add products to inventory
- Delete products from inventory
- View current inventory in a table
- See total products, total items, and total inventory price
- Save inventory data using localStorage
